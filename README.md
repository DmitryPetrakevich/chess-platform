# ♟️ Шахматная платформа
Веб-платформа для игры в шахматы с возможностью онлайн-игр, анализа партий и тренировки координат.

---

## ✨ Основные возможности

### 🎮 Игра с соперником
- Создание игр с гибким выбором контроля времени и цвета.
- Реальные игры через WebSocket соединение.
- Шахматный таймер.
- Предстартовый отсчет для первого хода.
  
---

### 👤 Авторизация и профиль
- Регистрация и вход в систему.
- Личный профиль с историей всех сыгранных партий.
- Просмотр сыгранных партий с возможностью перемотки по ходам.

---
### 📊 Просмотр партий
- Детальный просмотр любой сыгранной партии.
- Удобный реплеер с перемоткой по ходам.
  
---
### 🎯 Тренировка координат
- Специальный режим для развития навыков ориентации на доске.
- Возможность скрывать фигуры и надписи координат для тренировки памяти.
- Режимы игры:
  - Бесконечный режим.
  - Режим с таймером.
- Функции:
- Отображение текущей и следующей цели.
- Статистика по каждому цвету.
- Возможность переворачивать доску.
